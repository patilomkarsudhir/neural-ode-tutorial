# Neural ODE Quick Start Guide

## Setup Instructions

1. Install dependencies:
   `
   pip install -r requirements.txt
   `

2. Launch Jupyter:
   `
   jupyter notebook
   `

3. Open 
eural_ode_demo.ipynb and run all cells

## What You'll Learn

- How Neural ODEs work
- Training with the adjoint method
- Visualizing learned dynamics
- Testing extrapolation capabilities

## Expected Runtime

- CPU: 2-5 minutes
- GPU: <1 minute

Enjoy exploring Neural ODEs!
